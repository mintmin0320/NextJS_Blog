기존에는 react-speech-recognition 라이브러리를 사용해 회의(음성) 내용을 등록/수정/삭제할 수 있도록 구현했었는데, 1차 데모 발표를 끝내고 모든 기능들을 재정비하며 점검하던 중 회의록을 프린트할 수 있다면 좋겠다는 생각을 했다. 
역시나 NPM에는 없는 기능이 없다. 구글에 관련 키워드를 검색하여 곧바로 관련 NPM 문서를 발견할 수 있었다. 대략적으로 제공되는 샘플 코드를 봤을 때 큰 어려움이 없어 보였다. 곧바로 프로젝트에 적용해 보도록 하자.

<br/>

## 보일러 플레이트 코드

```
import React, { useRef } from 'react';
import { useReactToPrint } from 'react-to-print';

import { ComponentToPrint } from './ComponentToPrint';

const Example = () => {
  const componentRef = useRef();
  const handlePrint = useReactToPrint({
    content: () => componentRef.current,
  });

  return (
    <div>
      <ComponentToPrint ref={componentRef} />
      <button onClick={handlePrint}>Print this out!</button>
    </div>
  );
};
```

- useRef로 프린트하고 싶은 컴포넌트(ComponentToPrint)에 접근하고 useReactToPrint 메서드를 통해 프린트가 가능하다.
- onClick 이벤트가 발생하면 프린트될 컴포넌트의 CSS도 적용된다. 이 부분은 아래 사진에서 확인해 보자
